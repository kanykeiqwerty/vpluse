# Парсер данных 

**1. Создать виртуальное окружение и активировать:**

```
python3 -m venv venv
source venv/bin/activate
```

**2. Установить зависимости:**

```
pip install -r requirements.txt
```

**3. Запуск:**

```
python vpluse.py
```

**Результат:**
Создаётся `bankruptcy_data.xlsx` с листами:

* Юридические лица
* Физические лица
